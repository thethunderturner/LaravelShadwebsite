We're here once again with another update. Although we've been quite silent, that doesn't mean things weren't still moving along ;)

As stated in our release announcement, many things changed after 0.15.0, many of which created compatibility issues with previous versions. While unfortunate, we believe these changes were worth it for the new features we have been able to implement.

Here are some of the notable new features that the upcoming 0.16.0 version will bring:

 - Multi-controller support. ShadPS4 can now support up to 4 different controllers at a time.

 - Multiuser support. We now offer a user manager, which allows for creating up to 16 accounts (similar to the PS4).

 - Multilanguage trophies. Trophies now appear in the selected console language.

 - Big Picture Mode. A feature you've probably been seeing in other emulators as of late, is now available for shadPS4 as well.
