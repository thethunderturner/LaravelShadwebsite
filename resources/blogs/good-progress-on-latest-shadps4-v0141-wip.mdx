Although it is less that a week since our latest 0.14.0 release we already have made some progress on some parts of the emulator that increased compatibility in some games.

Fixing some stuff on libcinternal lib we can now have proper sound support in some games that appeared to be broken. Also , we rewrote audioOut interface and SDL3 backend. A OpenAL backend is also in works which will bring better audio3d support to the games that support it . We do also have some fixes on pad interface , fixing some bugs with a few games that didn't response to inputs.
